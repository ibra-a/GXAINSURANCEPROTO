# GXA Claims Platform - Talking Points
**Quick Reference for Presentation**

---

## Opening (30 seconds)

"Thank you for your time. Today I'm presenting the digital claims platform we've built for GXA. This system transforms how customers submit insurance claims - from paper forms to a modern mobile app that works on their phones. Let me show you what we've built."

---

## Key Message (1 minute)

**What It Does:**
- Customers submit claims from their phone
- Takes photos directly (like WhatsApp)
- Photos automatically tagged with GPS location (prevents fraud)
- Admin sees all claims instantly in dashboard
- Approve/reject with one click

**Why It Matters:**
- Faster: Claims processed in hours, not days
- Cheaper: Less admin work, fewer calls
- Secure: GPS prevents fake claims
- Professional: Matches GXA's reputation

## If Something Goes Wrong

**Camera doesn't open?**
→ "This requires HTTPS. On production deployment it works perfectly."

**Claim doesn't submit?**
→ "Let me check the console. This is why we test - catch issues early."

**Photo doesn't load?**
→ "May need to check storage permissions. Technical detail we'll resolve."

**Overall:**
Stay calm, acknowledge issue, explain it's a demo/testing environment, emphasize production will be perfect.
